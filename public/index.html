Simple CRUD Demo

Users dan Devices

Halaman sederhana untuk melihat data, cari data by id, dan tambah data baru.

Cara Pakai Halaman Ini

1. Lihat Semua Data

Klik Load Users atau Load Devices untuk mengambil data dari API dan menampilkannya ke halaman.

2. Cari Data By ID

Copy salah satu ID dari daftar data, lalu tempel ke form Cari User By ID atau Cari Device By ID.

3. Tambah Data Baru

Isi form Tambah User atau Tambah Device, lalu klik tombol simpan.

4. Lihat Hasil JSON

Setiap hasil request akan tampil di kotak hasil agar kamu bisa belajar bentuk response dari API.
